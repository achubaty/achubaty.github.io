more on journal abbreviations in Mendeley

Wow, my other posts on the topic are getting a lot of hits! Some exciting info for anyone still struggling with getting abbreviated journal names working with Mendeley: this feature is coming soon! This is currently on our roadmap for Mendeley Desktop 1.10. Expected release date is the start of August. From the mockups and descriptions the good folks at Mendeley have provided it looks like they’re doing a great job of addressing this shortcoming of their current product.

enabling Libre/OpenOffice plugin for Mendeley 1.5

I upgraded to using Mendeley v 1.5 (because previous versions don’t work on Ubuntu 12.04 Precise) and I had problems installing the OpenOffice plugin. Normally this is easy: in Mendeley you select the Tools menu and go to Install OpenOffice Plugin. However, for whatever reason the software can’t find the plugin, and I get this error: Mendeley couldn't locate unopkg in the default location. It should be in the 'program' subdirectory of the path where OpenOffice is installed.

However, one limitation I’ve encountered is that it does not have very many citiation styles preinstalled, and although many additional formats are available to download through the Mendeley interface, it does not (yet) provide a built-in style editor to customize citiation styles.
